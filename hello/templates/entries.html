Entry Form for Activities

City
Write down the name of the city


Activity/Venue Name
Fill in the name of the activity or venue as written in TripAdvisor, Google or any other source
Can be name of museum, restaurants, venues etc. or as listed on the source.


Activity Title
Fill in the title of the activity as you would like it to be displayed in the App.
Use Action Titles, make the titles interesting. Convert all foreign words to English.
Capture the essence of what you can do at the venue in less than 8 words.


Filter Hashtag
Select the hashtag which most corresponds to the type of activity.


Activity Description
Fill in the description of the activity including what can be done in the activity. An extension of activity title.
Do not use more than two lines


Image Link
Link to an image which captures the essence of the activity. Use Google Images wherever preferred.


Recommended time for activity (in minutes)
Recommended time to spend on activity in minutes.


Rating
If the activity has a TripAdvisor or any other source rating, fill it here.


Price
Enter lowest possible price of Activity, if free, then enter 0.


Compulsory Pre-book
Select Yes if an advance reservation is required to do this activity. If optional reservation or no reservation, select No.


Deposit
Enter advance deposit to be paid (if any) for the activity. If not applicable, leave blank


Time Availability type
Select the availability type which most corresponds to the type of activity.


Timings
Enter timings of the activity. (Please fill in format taking care of spaces)

Format:
Unrestricted activities - Do not enter anything.

Regular activities - Days for which activity open seperated by ", " (Write Daily if open all days) " - " Time when open (24hr format) "to" Time when close (24 hr format)
e.g. Monday, Tuesday, Wednesday, Friday, Saturday - 9 to 18 or Daily - 8:30 to 20:30

Slots - Days for which activity open seperated by ", " (Write Daily if open all days) " - " Possible reservation times seperated by ", " (24 hr format)
e.g. Daily - 9, 10, 11, 12, 14, 15 or Monday, Tuesday, Wednesday - 10, 11:30, 14

Occurence - Full date (in format DD Full_Month_Name Year) " - " Time when occuring (24hr format). In case of multiple occurences, separated by ", "
e.g. 18 August 2017 - 20:30, 20 September 2017 - 19

Duration - Start Date (in format DD Full_Month_Name Year) " to " End Date (in format DD Full_Month_Name Year) " - " Start Time (24 hr format) " to " End Time (24 hr format)
e.g. 17 July 2017 to 18 July 2017 - 9 to 17


Provider
Fill in name of provider who provides the activity. Majorly applicable for Adventure Sports and Events. Leave blank if not known.


Website for booking
Enter website of provider from where the activity can be booked.
In case of multiple activities from the same provider, fill in a common page from where all the activities can be accessed.
